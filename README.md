# life.txt
An sh program that manages notes

This is just a small first project while learning the Shell scripting languaje.

ROADMAP:
- Creates notes
- Read notes
- Edit notes
- Delete notes
- List notes

- Creates tasks
- Read tasks
- Edit tasks
- Check tasks
- Delete tasks
- List tasks
- Tasks priorities

- Creates events in a calendar
- Read events
- List events (weekly or monthly)
- Edit events
- Delete events
- "Checks" passed events automatically
